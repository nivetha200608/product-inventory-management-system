# Product Inventory Management System

## Project Description
The Product Inventory Management System is a web application developed to manage product details in an inventory.  
This system allows users to register, login and perform product management operations like adding, viewing, updating and deleting products.

## Features
- User Registration
- User Login
- Add Product
- View Products
- Update Product
- Delete Product

## Technologies Used
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Project Structure
- models – Database models
- routes – API routes
- middleware – Authentication middleware
- config – Database connection
- server.js – Main server file

## How to Run the Project
1. Clone the repository  
2. Install dependencies using `npm install`  
3. Start the server using `npm start` or `nodemon server.js`  
4. Open Postman and test the APIs  
5. Server will run on `http://localhost:5000`
